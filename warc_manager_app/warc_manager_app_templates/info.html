Information

This webapp will allow the public to peruse Brown Library WARC web-captures.

The WARC (Web ARChive) file format is an archival standard used to store web crawls, which include website data and metadata. It is widely used in web archiving projects, such as the Internet Archive's Wayback Machine, national libraries, and other institutions that preserve web content for historical or research purposes.

This webapp also offers features for Library staff to manage WARC files.
